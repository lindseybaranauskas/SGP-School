Executive Overview

Plan how new growth is absorbed into the VP leadership network.

This portal compares the current VP assignment structure against optimized scenarios that balance workload capacity, geography, EVS/CNS alignment, facility complexity, and implementation disruption.

      Model Assumptions

      • The dashboard is a decision-support tool, not an automatic final assignment approval system.
      • Capacity, distance, EVS/CNS fit, complexity, and reassignment disruption are treated as tradeoffs.
      • Assignments marked for review should be validated before implementation.
      • Network views are intended to show relationship structure and concentration patterns.
